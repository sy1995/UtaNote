# UtaNote

日本动漫歌曲歌词解析工具 - 将动漫歌曲转换为带词语解析、语法标注的学习资料。

## 功能特点

- 智能搜索：通过模糊关键词（如"鬼灭之刃 OP"）自动识别歌曲
- 完整歌词：获取日文、罗马音、中文翻译三对照歌词
- 词语解析：逐词拆分，标注假名、罗马音、词性、JLPT 等级
- 语法注释：详细解释助词用法、动词变形等语法点

## 快速开始

### 1. 安装依赖

```bash
# 创建虚拟环境（推荐）
python3 -m venv .venv
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 2. 配置 API Key

```bash
cp .env.example .env
# 编辑 .env 文件，填入你的阿里云 DashScope API Key
```

### 3. 生成歌曲

```bash
# 命令行方式
python3 generate_song.py "鬼灭之刃 OP"
python3 generate_song.py "红莲华 LiSA"

# 交互式方式
python3 generate_song.py
```

## 工作流

```
│  1. 搜索    │ ──► │  2. 确认    │ ──► │  3. 解析    │ ──► │  4. 保存    │
│  歌曲信息   │     │  歌词内容   │     │  词语语法   │     │  JSON 文件   │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

### 步骤说明

1. **搜索阶段**：调用大模型联网搜索，识别歌曲并获取完整歌词（日文 + 罗马音 + 中文）
2. **确认阶段**：预览歌词前 10 行，确认无误后继续
3. **解析阶段**：逐行调用大模型，拆分词语并标注语法
4. **保存阶段**：生成 JSON 文件到 `output/` 目录

## 贡献歌曲

欢迎提交更多动漫歌曲！

### 提交流程

1. **生成歌曲 JSON**
   ```bash
   python3 generate_song.py "歌曲名"
   ```

2. **检查输出文件**
   - 确认 `output/` 目录下生成了正确的 JSON 文件
   - 检查歌词、词语解析是否准确

3. **提交 Git**
   ```bash
   git add output/your_song.json
   git commit -m "add: 歌曲名 - 歌手名"
   git push
   ```

### 提交规范

- **Commit 格式**：`add: 歌曲名 - 歌手名`
- **文件命名**：使用歌曲 ID（小写英文/罗马音），如 `gurenge.json`
- **必填字段**：确保 JSON 包含所有必填字段（id, title, artist, anime, type, lines）

### 示例

```bash
# 生成歌曲
python3 generate_song.py "One Piece OP1 We Are"

# 检查结果
ls output/

# 提交
git add output/we_are.json
git commit -m "add: We Are - きただにひろし"
git push origin main
```
